# barsrevamp
How to test it locally in Roblox Studio

Open your Roblox place in Studio.
In Explorer, open StarterGui (or StarterPlayerScripts if you prefer).
Create a new LocalScript and paste the contents of src/StatsHUD.local.lua into it (or copy the file from the repository into StarterGui).
Play (Start > Play) to run the local client:
Press H, J, K, L to simulate stat changes and see animations.
Hold LeftShift to drain stamina and release to trigger regen behavior.
